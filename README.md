A modern, responsive Single-Page Application (SPA) showcasing the portfolio and case studies of **Frontend Software Engineer, Anfisa Domashova**. Built using **React, Framer Motion, and Tailwind CSS** for high performance and maintainability.

---

## Features

- Responsive, mobile-first layout using **Tailwind CSS**
- Smooth animations powered by **Framer Motion**
- Reusable `UniversalModal` component
- Simple data-driven projects setup
- Accessibility-friendly external links

---

## Prerequisites

- Node.js 16+ or current LTS
- npm or yarn

---

## Quick Start

1. Clone repository  
2. Install dependencies  
   - `npm install` or `yarn`  
3. Start dev server  
   - `npm run dev` or `yarn dev`  
4. Build for production  
   - `npm run build` or `yarn build`  
5. Preview build  
   - `npm run preview` or `yarn preview`
